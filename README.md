* Web App ERP for small company, notably machine builders.
* It was developed for a previos comapny I worked for. I have been using to track and control my own side projects.
* It uses ASP.net/Framework, SQL, and HMTL5 technologies.
* It has modules for inventory control, timetracker, invoicing, project management, purchase orders and bill of materials.

 [Webapp ERP](https://lrerp.azurewebsites.net/).
